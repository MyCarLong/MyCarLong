## 4. Installation
### Requirements
- Spring Boot 3.x
- Java 17 or higher
- [Node.js](https://nodejs.org/en/download) 14 or higher
- JavaScript ES6

### Getting Started

```bash
$ git clone https://github.com/your-username/MyCarLong.git
$ cd MyCarLong
```

### Backend
```bash
$ cd backend
$ ./gradlew bootRun
```
### Frontend
```bash
$ cd frontend
$ npm start
```
